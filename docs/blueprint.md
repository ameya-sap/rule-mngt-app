# **App Name**: RuleMaster

## Core Features:

- Rule Creation: Intuitive interface for creating business rules with conditions and actions.
- Rule Listing: Comprehensive list view of all business rules.
- Rule Editing: Detailed view with rule editing capabilities.
- Rule Activation: Simple toggle for business rule activation/deactivation.
- Rule Suggestion: AI-powered "tool" to suggest possible conditions and actions for rules based on existing rules and patterns.
- Rule Import: Import rules from external sources like CSV or JSON files.
- Rule Export: Export rules to share with colleagues.

## Style Guidelines:

- Primary color: Professional blue (#3498DB) to convey trust and reliability.
- Background color: Light blue (#EBF5FB) to maintain a clean and modern look.
- Accent color: A slightly darker shade of blue (#2980B9) to highlight important interactive elements.
- Body and headline font: 'Inter', a sans-serif font, providing a modern and neutral feel suitable for both headlines and body text.
- Use simple, outline-style icons to represent different rule actions and conditions, keeping the interface clean.
- Employ a clean and structured layout with clear visual hierarchy, ensuring easy navigation and comprehension of the rules.
- Subtle animations for state changes (e.g., rule activation) to provide visual feedback without being distracting.
